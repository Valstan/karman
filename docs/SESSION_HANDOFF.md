## Failed approaches (этой нитки)

- **`endUntil: optionalDateString` без `.optional()`** — на Zod v4 union-с-`z.undefined()` НЕ
  делает ключ опциональным (в v3 делал); форма опускает ключ при `endType:'never'` → весь
  create-путь повторов падал «expected nonoptional». Юнит-зелёный не ловил (нет теста на
  absent-ключ); поймано G54 live-round-trip. Фикс — явный `.optional()` + регрессии.
- **Возврат «первого кандидата > now» в движке при включённых сдвигах** — тихие часы/будни
  могут переставить соседние кандидаты местами, поэтому со сдвигами берём МИНИМУМ инстанта по
  всем кандидатам (сдвиги только вперёд); быстрый путь без сдвигов сохранён неизменным.
